# Boiling Points Estimator



Estimate boiling points (b.p.) of molecules for easier distillation.



It follows the Clausius-Clapeyron equation assuming ΔH is constant. For detail : [Goodman et al. Tetrahedron Letters 2000, 41 (50), 9879–9882.](https://doi.org/10.1016/s0040-4039(00)01754-8.)



Deployed at [https://estimate-bp.streamlit.app/](https://estimate-bp.streamlit.app/)
